# 🌡️ Temperature Converter

A modern, responsive, and user-friendly Temperature Converter built using **HTML**, **CSS**, and **JavaScript** as part of the **SkillCraft Technology Software Development Internship - Task 1**.

---

## 📌 Project Overview

This web application allows users to convert temperatures between:

- Celsius (°C)
- Fahrenheit (°F)
- Kelvin (K)

The application features a modern Glassmorphism UI, responsive layout, live conversion, and input validation.

---

## 🚀 Features

- ✅ Celsius ↔ Fahrenheit ↔ Kelvin Conversion
- ✅ Live Temperature Conversion
- ✅ Input Validation
- ✅ Swap Units
- ✅ Reset Button
- ✅ Keyboard Support (Press Enter)
- ✅ Responsive Design
- ✅ Modern Glassmorphism UI
- ✅ Smooth Animations
- ✅ Mobile Friendly
